Usability Issues with Expense Tracking Spreadsheet

Learnability

Issue: Complex Formula Usage - If the spreadsheet relies heavily on complex formulas or calculations that users aren't familiar with, it can be challenging for Neo and Trinity to learn how to use and modify the spreadsheet effectively.

Impact: This affects learnability, as it increases the learning curve and can lead to errors if not used correctly.

Efficiency

Issue: Manual Calculation Errors - Manually calculating amounts owed and inputting them into the spreadsheet can be time-consuming and error-prone. This is especially true as the number of transactions increases.

Impact: This affects efficiency, as it requires more effort and time to perform tasks correctly.

Safety

Issue: Lack of Data Validation - Without proper data validation, users might input incorrect amounts or descriptions, leading to inaccurate calculations or confusion.

Impact: This affects safety, as it increases the likelihood of errors and makes it harder to recover from mistakes.

Efficiency

Issue: Cumbersome Tracking - Over time, as the number of transactions accumulates, the spreadsheet might become difficult to navigate and manage, potentially leading to missed or duplicate entries.

Impact: This affects efficiency, as it slows down the process of inputting, reviewing, and settling transactions.

Learnability

Issue: Lack of User-Friendly Interface - If the spreadsheet lacks a clear and intuitive interface, new users might struggle to understand how to input data and navigate the various columns and categories.

Impact: This affects learnability, as it adds complexity to the process of using the spreadsheet.

Efficiency

Issue: Difficulty in Spotting Anomalies - Identifying anomalies (transactions that deviate from the norm) might be challenging, especially as the spreadsheet grows. This could lead to inaccurate tracking of shared expenses.

Impact: This affects efficiency, as it slows down the process of identifying and addressing anomalies.

Safety

Issue: Data Loss Risk - If the spreadsheet is not backed up properly, there's a risk of data loss due to technical issues or accidental deletion.

Impact: This affects safety, as it jeopardizes the integrity of the financial records.

To address these usability problems and improve the overall experience, Neo and Trinity might consider transitioning to a dedicated financial management app or tool designed for couples. These tools often offer automated expense tracking, user-friendly interfaces, automatic calculations, and data validation, addressing many of the issues associated with using spreadsheets for this purpose.
